# Wrestle Jump

一个基于原生 JavaScript + Canvas 的双人本地对战小游戏。

## 功能特点

- 双人同屏对战（玩家1 / 玩家2）
- 按住按键蓄力，松开释放跳跃
- 头部触地或撞到左右边界判负
- 回合制计分，先到 5 分获胜

## 运行环境

- Node.js 16+（推荐 LTS 版本）

## 快速开始

```bash
npm install
npm start
```

启动后访问：`http://localhost:3000`

## 操作说明

- 玩家 1：`A`（按住蓄力，松开起跳）
- 玩家 2：`L`（按住蓄力，松开起跳）
- `R`：重置整场比赛（分数清零）

## 胜负规则

- 任一玩家头部碰到地面，或头部撞到左右墙面，该玩家本回合失败
- 对手获得 1 分
- 某位玩家达到 5 分后，整场比赛结束

## 项目结构

- `index.html`：页面与基础样式
- `main.js`：游戏核心逻辑（物理、渲染、输入、计分）
- `server.js`：本地静态文件服务器
- `package.json`：项目脚本与元信息

## 开发命令

```bash
npm run dev
```

当前与 `npm start` 一致，均启动本地服务器。
